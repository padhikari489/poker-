# Poker++
A Poker game written in C++ by Pawan Adhikari, Aashish Giri, and Antony Nguyen
### Prerequisites
This game requires a Linux machine, as well as the Boost library, installed by:
```
sudo apt-get install libboost-dev
```
## Installation
To compile, execute:
```
make
```
### Execute the server
```
./dealer <port>
```
### Execute client(s)
```
./player <host-ip> <port>
```
